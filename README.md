## HACYBERGLOBALTECH
Official automation gateway for HGT-BOT - Activation Portal
HGT Bot is a secure, automated lead generation and verification system. This portal allows users to initiate service activation requests, which are processed in real-time by a backend Telegram bot engine.
# 🚀 Features
Web Portal: A sleek, mobile-responsive interface for capturing user information.
 Real-time Notifications: Instant lead forwarding to administrative chats via Telegram.
Verification Workflow: State-managed bot logic to handle manual verifications, receipt uploads, and status tracking.
 Secure Infrastructure: Token-based authentication using environment variables.
# 🛠 Tech Stack
 Frontend: HTML5, CSS (Custom styling)
 Backend: Node.js, Telegraf.js
 Deployment: GitHub Pages (Frontend) & Render/Railway (Backend)
